HTML Tags References

Document Metadata

These contain information about the page needed by software to render the page.

Tag Name Description
<head> Head Contains information about the document, like it's title, scripts and style sheets
<link> Link Creates a link between the HTML document and an external resoucre.
<style> Style Contains CSS, which is applied to the document.

Content Sectioning

Content sectioning allows you to organise your content into logical pieces.

Tag Name Description
<body> Body Contains information about all the contents.
<header> Heading section Contains introductory information, which may include logo, navigation or search bar.
<main> Main section Contains the dominant content of the body of the document.
<h1>, <h2>, <h3>,
<h4>, <h5>, <h6>		 Section headings Used to create hierachy in headings.
<nav> Navigation Creates a section of the page that provides nagivation links, either within the document or to external links
<section> Section Creates a standalone section

Text Content

Use HTML text content elements to organize blocks or sections of content placed between the body tags.

Tag Name Description
<div> Content division A generic container for flow content.
<li> List Item Creates an item within a list. Contained within <ol> or <ul>.
<ol> Numbered List Creates a list that is numbered.
<ul> Un-Numbered List Creates a list that is un-numbered.
<p> Text Used to insert typed text.
